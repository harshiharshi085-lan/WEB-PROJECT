Event Booking Website

A modern and responsive Event Booking Website developed using HTML5 and CSS3. This project is designed to provide users with an easy and interactive platform to explore events, view upcoming programs, and book tickets online.

The website includes a clean navigation system, visually attractive event cards, smooth scrolling effects, and a user-friendly contact form. It showcases different event categories such as music concerts, art exhibitions, food festivals, tech conferences, and entertainment shows.

This project demonstrates the use of:

Responsive web design
Flexbox layouts
CSS hover effects
Sticky navigation bar
Image handling with object-fit
Structured webpage sections

The application is divided into multiple sections including:

Home – Displays featured events and banner
Events – Shows upcoming events with booking options
About – Provides project and team information
Contact – Allows users to send messages through a form

Technologies Used

CSS3

Features
