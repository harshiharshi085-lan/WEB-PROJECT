#  Event Booking Website

A responsive and user-friendly event booking website built using HTML and CSS. The project provides a clean interface for users to explore featured events, view upcoming programs, and access booking options through an organized and visually appealing layout.

The website includes multiple sections such as Home, Events, About, and Contact, along with a sticky navigation bar and smooth scrolling for improved user experience. Event cards are designed using Flexbox to maintain consistent alignment and responsive structure across different screen sizes.

This project focuses on frontend web development concepts including responsive design, layout structuring, image handling, hover effects, and modern UI styling using pure CSS.

### Key Features

* Responsive webpage layout
* Sticky navigation bar
* Smooth scrolling navigation
* Event showcase cards
* Interactive hover effects
* Contact form interface
* Clean and modern UI design

### Technologies Used

* HTML5
* CSS3

### Purpose

This project was developed to demonstrate frontend web design skills and create a simple event management interface with an attractive and structured user experience.
